# .sim File Generator
A python program that generates a .sim file from a input boolean expression. 
**Gates identified by the program**
 - NOT Gate: ~
 - AND Gate: &
 - OR Gate: |
 - XOR Gate: ^
 - NAND Gate: ~&
 - NOR Gate: ~|
 - XNOR Gate: ~^

## Prerequisites
```bash
$ sudo apt-get update
$ sudo apt-get install python3.6
```
